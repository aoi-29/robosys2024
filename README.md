# Reverse Text Tool
![test](https://github.com/aoi-29/robosys2024/actions/workflows/test.yml/badge.svg)

## 概要
標準入力から受け取ったテキストを反転して表示する。

## 必要なソフトウェア
- Python
  -テスト済みバージョン：　3.12.3

## テスト環境
- Ubuntu 24.04 LTS

## 使用方法
- `$ echo "テキスト" | ./reverse_text`

このソフトウェアパッケージは、３条項BSDライセンスの下、再頒布および使用が許可されます。
